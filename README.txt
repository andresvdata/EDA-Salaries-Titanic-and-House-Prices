This project is for assignment 3 for the class CSPC at Langara College.
Exploratory Data Analysis will be applied to two datasets:
* Salaries
* Titanic
* House Prices



There are 4 folders in this project:
* Src: Jupyter notebooks will be saved here!
	Assignment_3_Part_1: EDA for Salaries data.
	Assignment_3_Part_2: EDA for Titanic data.
	Assignment_3_Part_3: EDA for House Prices data.
* Results: The HTML versions of Jupyter notebook will be saved here!
* Data: All the data used in this project will be found here!
* Docs: In this folder will be saved PDF version for all the Jupyter Notebooks.
